# TypeScript安装和使用

<code>npm install typescript -g</code> 全局安装

使用有两种方式:

+ tsc直接编译一个<code>.ts</code>文件，生产<code>.js</code>文件
+ 在项目中编译，通过<code>npm run tsc</code>实现对整个项目编译

在ts-practice文件夹下新建一个src目录，src目录下新建一个index.ts文件

index.ts里写入如下内容:

```typescript
export enum TokenType {
    ACCESS = 'accessToken',
    REFRESH = 'refreshToken'
}
```

用<code>npm init</code>初始化项目，生成了<code>package.json</code>文件，在<code>package.json</code>文件的scripts标签里写入：

```json
"scripts": {
    "test": "echo \"Error: no test specified\" && exit 1",
    "tsc": "tsc"
  }，
"main": "src/index.ts"
```

然后，用<code>tsc --init</code>命令进行初始化
